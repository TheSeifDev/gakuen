Q1: Functions with Loops
المطلوب: اكتب دالة برمجية باسم count_even_numbers(numbers).
يجب أن تستقبل هذه الدالة "قائمة من الأرقام"، وتقوم بإرجاع عدد الأرقام الزوجية الموجودة داخل القائمة.
  • Write a function count_even_numbers(numbers).
  • Input: List of integers.
  • Returns: Count of even numbers.
  • Example: [1, 2, 3, 4, 6] -> Returns 3
solution.py
Model Answer: 
def count_even_numbers(numbers):
    count = 0
    for num in numbers:
        if num % 2 == 0:
            count += 1
    return count

Logic & Explanation:

We initialize a count variable to 0. We iterate through every number in the list. The modulo operator % 2 == 0 checks if a number is even.
الشرح: نبدأ بتعريف متغير count بقيمة صفر. نستخدم حلقة تكرار للمرور على كل عنصر في القائمة. المعادلة % 2 == 0 هي الطريقة البرمجية للتحقق من أن الرقم زوجي.
Q2: Functions with Conditions
المطلوب: اكتب دالة check_grade(score) للتحقق من الدرجات (0-100).
يجب أن تُرجع التقدير نصياً: "Excellent" (للدرجة 85 فأكثر)، "Very Good" (75)، "Good" (65)، "Pass" (50)، وفي غير ذلك تُرجع "Fail".
  • Write a function check_grade(score) (0-100).
  • Returns "Excellent" (≥85), "Very Good" (≥75), "Good" (≥65), "Pass" (≥50), else "Fail".
grades.py
Model Answer: 
def check_grade(score):
    if score >= 85:
        return "Excellent"
    elif score >= 75:
        return "Very Good"
    elif score >= 65:
        return "Good"
    elif score >= 50:
        return "Pass"
    else:
        return "Fail"

Logic & Explanation:

We use an if-elif-else chain. It checks conditions from highest to lowest. As soon as one condition is true, Python executes it and ignores the rest.
الشرح: نستخدم سلسلة الشروط if-elif-else. يتم فحص الشروط بالترتيب من الأعلى للأسفل. بمجرد أن يتحقق شرط واحد (مثلاً الدرجة أكبر من 85)، تتوقف الدالة وتعطي النتيجة فوراً.
Q3: Functions with Lists and Tuples
المطلوب: اكتب دالة min_max_tuple(numbers).
المهمة هي إيجاد أصغر وأكبر رقم في القائمة، وإرجاع الناتج داخل Tuple بالشكل التالي: (أصغر رقم، أكبر رقم).
  • Write min_max_tuple(numbers).
  • Return a tuple: (min_value, max_value).
  • Example: [4, 9, 1, 7] -> (1, 9)
tuples.py
Model Answer: 
def min_max_tuple(numbers):
    minimum = numbers[0]
    maximum = numbers[0]
    for num in numbers:
        if num < minimum:
            minimum = num
        if num > maximum:
            maximum = num
    return (minimum, maximum)

Logic & Explanation:

We assume the first number is both the min and max. Then we loop through the rest. If we find a smaller number, we update min. If we find a larger one, we update max.
الشرح: نفترض مبدئياً أن أول رقم هو الأصغر والأكبر. ثم نقارنه بباقي الأرقام؛ إذا وجدنا رقماً أصغر نُحدث قيمة الـ min، وإذا وجدنا أكبر نُحدث الـ max. في النهاية نُرجع النتيجة بين أقواس () لإنشاء Tuple.
Q4: Functions with Sets
المطلوب: اكتب دالة common_elements(list1, list2).
قم بتحويل القائمتين إلى Sets (مجموعات) ثم أرجع العناصر المشتركة فقط (التقاطع) بينهما.
  • Write common_elements(list1, list2).
  • Convert lists to sets and return the intersection (common elements).
sets.py
Model Answer: 
def common_elements(list1, list2):
    set1 = set(list1)
    set2 = set(list2)
    return set1 & set2

Logic & Explanation:

Lists allow duplicates, Sets do not. Converting to sets removes duplicates. The & operator is the standard mathematical intersection operator for sets.
الشرح: القوائم تسمح بالتكرار، لكن المجموعات (Sets) لا تسمح بذلك. نحول القوائم إلى Sets باستخدام set(). الرمز & يقوم بعملية التقاطع الرياضية لإيجاد العناصر الموجودة في كلا المجموعتين.
Q5: Modules (Main.py)
المطلوب:
1. اكتب كود لاستيراد الملف المسمى my_functions واستدعاء الدوال التي كتبتها سابقاً.
2. استورد مكتبة بايثون الرياضية math واستخدم دالة الجذر التربيعي sqrt.
  • Part A: Write code to import my_functions and call the previous functions.
  • Part B: Import math and use sqrt.
main.py
Model Answer: 
import my_functions
import math

# Part A
print(my_functions.count_even_numbers([1, 2, 6]))
print(my_functions.check_grade(78))

# Part B
print(math.sqrt(25))

Logic & Explanation:

The import statement allows you to use code from other files or libraries.
الشرح: الأمر import هو المفتاح لاستخدام أكواد خارجية. my_functions هو ملف قمت أنت بإنشائه، بينما math هي مكتبة جاهزة تأتي مع بايثون للعمليات الحسابية المعقدة.

lists.py
Model Answer:

Section 4: Functions & Modules

Q12: Functions with Loops
المطلوب: اكتب دالة برمجية باسم count_even_numbers(numbers).
يجب أن تستقبل هذه الدالة "قائمة من الأرقام"، وتقوم بإرجاع عدد الأرقام الزوجية.
  • Write a function count_even_numbers(numbers).
  • Input: List of integers.
  • Returns: Count of even numbers.
solution.py
Model Answer: 
def count_even_numbers(numbers):
    count = 0
    for num in numbers:
        if num % 2 == 0:
            count += 1
    return count

    if score >= 85:

    elif score >= 75:

    elif score >= 65:

        return "Pass"

Q14: Functions with Lists and Tuples
المطلوب: اكتب دالة min_max_tuple(numbers) لإيجاد أصغر وأكبر رقم.
يجب إرجاع الناتج داخل Tuple: (أصغر رقم، أكبر رقم).
tuples.py
Model Answer: 
def min_max_tuple(numbers):
    minimum = numbers[0]
    maximum = numbers[0]
    for num in numbers:
        if num < minimum:
            minimum = num
        if num > maximum:
            maximum = num
    return (minimum, maximum)
Q15: Functions with Sets
المطلوب: اكتب دالة common_elements(list1, list2).
حول القوائم إلى Sets وأرجع التقاطع (العناصر المشتركة).
sets.py
Model Answer: 
def common_elements(list1, list2):
    set1 = set(list1)
    set2 = set(list2)
    return set1 & set2

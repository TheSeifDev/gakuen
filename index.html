Assignment 04: Loops (For & While)

Q1. Write a Python program that prints the letters of the “University” keyword except the “v” letter.
الترجمة: اكتب برنامج بايثون لطباعة حروف كلمة "University" باستثناء حرف "v".
الشرح: نستخدم حلقة التكرار (for loop) مع جملة شرطية (if) لاستثناء الحرف.
main.py Python
Solution: 
for letter in "University":
    if letter == "v":
        continue
    print(letter)
Q2. Write a Python program that prints numbers from 0 to 5 several times, where you need to read the number of times from the user.
الترجمة: برنامج لطباعة الأرقام من 0 لـ 5 عدة مرات بناءً على طلب المستخدم.
الشرح: نحتاج إلى حلقتين متداخلتين (Nested Loop)؛ الخارجية للتكرار والداخلية للأرقام.
main.py Python
Solution: 
n = int(input("Enter times: "))
for i in range(n):
    for j in range(6):
        print(j, end=" ")
    print() # New line
Q3. Write a Python program that prints even numbers less than 20.
الترجمة: اكتب برنامج لطباعة الأعداد الزوجية الأقل من 20.
الشرح: يمكن استخدام دالة range مع خطوة (step) 2، أو استخدام عملية باقي القسمة %.
main.py Python
Solution: 
# Method 1
for i in range(0, 20, 2):
    print(i)

# Method 2
x = 0
while x < 20:
    print(x)
    x += 2
